#  TwitchTTS Chickens — Интерактивные TTS-Питомцы для Minecraft 1.21

[![Minecraft](https://img.shields.io/badge/Minecraft-1.21-5D9148?style=for-the-badge&logo=minecraft)](https://www.minecraft.net/)
[![Fabric](https://img.shields.io/badge/Loader-Fabric-e07e26?style=for-the-badge)](https://fabricmc.net/)
[![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)

Преврати чат твоего Twitch-стрима в полноценную RPG-механику! Этот мод добавляет умных питомцев (кур), которые появляются в мире, когда зрители пишут сообщения в чат. Они озвучивают сообщения, защищают игрока от мобов, фармят ресурсы и эволюционируют прямо на глазах, а чат может напрямую влиять на игровой процесс с помощью специальных команд.

---

## ✨ Основные возможности

- ️ **Голосовая озвучка**: Сообщения из чата автоматически добавляются в очередь для качественной TTS-озвучки.
- ⚔️ **Боевая система**: Курицы самостоятельно атакуют враждебных мобов в радиусе 12 блоков.
- 🛡️ **Умный режим "Телохранитель"**: Если игрока атакуют, все курицы в радиусе 15 блоков впадают в ярость (получают Скорость II и Силу II) и бросаются на защиту. *(Игнорируют урон от падения, огня и магии, чтобы не реагировать ложно)*.
-  **Система эволюции**: 1 сообщение в чате = 1 уровень (максимум 50 уровень). Визуальные эффекты меняются: свечение (5 ур.), частицы чар (10 ур.), огненная аура (30 ур.), огонь душ и утроенный урон (50 ур.).
- 🍖 **Голод и Привязанность**: За питомцами нужно ухаживать. Если голод и привязанность упадут до критических значений, курица может убежать.
- 🎁 **Эпический дроп**: Курицы высокого уровня периодически дарят игроку ценные предметы (от железа до Элитр с фейерверками!).

## 🎮 Команды

### ⚙️ Команды для игрока (в игре):
- `/twitchtest <текст>` — Проверить работу TTS очереди.
- `/fun true` или `/fun false` — Включить или выключить режим хаоса для команд чата.
- `/chickens` — Показать подробную таблицу со статистикой всех активных кур (уровень, голод, привязанность, время жизни).
- `/chicken_immortal` — Сделать текущую курицу бессмертной (отключает таймер жизни).
- `/tts help`, `/tts status`, `/tts clear` — Управление базовыми функциями мода.

### 📺 Команды для зрителей (в чате Twitch):
*Все команды имеют встроенные кулдауны для предотвращения спама.*

**Базовые взаимодействия:**
- `!feed` — Покормить всех куриц стримера (+20 голода, кулдаун 30 сек).
- `!pet` — Погладить всех куриц стримера (+5 привязанности, кулдаун 10 сек).
- `!drop` — Эпический дроп предметов рядом со стримером (от железа до Незерита и Элитр с фейерверками!).
- `!hit` — Отбросить стримера.
- `!effect` — Случайный положительный или отрицательный эффект для стримера.
- `!levitate`, `!speed`, `!blindness` — Временные эффекты для стримера.
- `!day` / `!night` — Сменить время суток.
- `!creeper`, `!cat`, `!dog` — Призвать существо рядом со стримером.

**⚠️ Режим ХАОСА (работает ТОЛЬКО если стример ввел `/fun true`):**
- `!void` — Вырыть яму 6x6 глубиной 20 блоков под стримером.
- `!meteor` — Метеоритный дождь вокруг стримера.
- `!wither` — Призвать Визера.
- `!tnt_rain` — Дождь из активирующегося динамита.
- `!earthquake` — Землетрясение, тошнота и призыв зомби.
- `!rain_frogs` — Дождь из лягушек.
- `!phantom` — Призвать стаю фантомов.
- `!swap` — Поменять местами стримера и ближайшего моба.
- `!freeze` — Заморозить стримера (слепота + замедление).
- `!thunder` / `!lightning` — Вызвать грозу или удары молний.
- `!anvil` — Уронить наковальню на голову стримера.

## 🛠 Установка

1. Установи **Minecraft 1.21** и загрузчик модов **Fabric** (рекомендуется через Prism Launcher или Modrinth App).
2. Скачай файл мода **TwitchTTS** и обязательный **Fabric API** из раздела [Releases](https://github.com/yungdv/twitchtts/releases).
3. Помести `.jar` файлы в папку `mods` твоего профиля Minecraft.
4. *(Рекомендуется)* Установи моды **Entity Model Features (EMF)** и **Entity Texture Features (ETF)**, если используешь ресурспак *Fresh Animations*, чтобы куры выглядели корректно и с анимациями.
5. Запусти внешний `TTS_Server.exe` (или аналогичный скрипт из архива) для обработки сообщений и их озвучки.
6. Зайди в игру → **Mod Menu** → **TwitchTts** → впиши имя своего Twitch-канала и настрой остальные параметры.
7. Зайди в мир и напиши `/twitchtest привет` для проверки работы.
