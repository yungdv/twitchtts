# 🐔 TwitchTTS - Интерактивные TTS Курицы для Minecraft

Мод для Minecraft 1.21 (Fabric), который добавляет интерактивных куриц-питомцев, читающих чат Twitch голосом, защищающих игрока и фармящих лут.

![Fabric](https://img.shields.io/badge/Fabric-1.21-blue)

## ✨ Фичи

-  **TTS Озвучка** - Курицы читают чат Twitch голосом
- ⚔️ **Боевая система** - Курицы атакуют враждебных мобов
- ️ **Режим телохранителя** - Курицы приходят в ярость, когда игрока бьют
-  **Система прогрессии** - 4 уровня эволюции (🐔 →  → ⭐ → 👑)
- 🎮 **Интерактив** - Зрители могут управлять курицами через команды
- 🎁 **Эпический дроп** - Шанс получить элитры, тотемы, алмазы
- 🍇 **Кастомные предметы** - Ягода Твича, Монета Стримера, Корона Стримера

## 🎮 Команды для зрителей

- `!drop` - Случайный предмет (кулдаун 30 сек)
- `!heal` - Лечение стримера (кулдаун 1 мин)
- `!pet` - Погладить курицу (кулдаун 30 сек)
- `!dance` - Танец курицы (кулдаун 30 сек)
- `!levitate` - Левитация стримера (кулдаун 3 мин)
- `!gamble` - Казино 50/50 (кулдаун 2 мин)

## 🛠 Установка

1. Скачай последний релиз из раздела [Releases](https://github.com/yungdv/twitchtts/releases)
2. Следуй инструкции в архиве
3. Запусти `TTS_Server.exe`
4. Настрой ник Twitch канала в меню мода (Mod Menu → TwitchTts → Config)
